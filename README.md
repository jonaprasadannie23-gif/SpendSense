# 💰 SpendSense

A Personal Finance Management Application built using React, Flask, and SQLite.

## 🌐 Live Demo

https://spend-sense-peach.vercel.app

## ✨ Features

- User Registration & Login
- Password Validation
- Expense Tracking
- Monthly Expense Analytics
- Income Management
- CSV Report Export
- Dark / Light Theme Toggle
- Responsive Dashboard

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- CSS
- Axios

### Backend
- Flask
- SQLite

### Deployment
- GitHub
- Vercel

### Frontend


### Backend

```bash

python app.py
```
